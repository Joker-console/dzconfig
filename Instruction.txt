Инструкция по использованию программы translator.py
Назначение:

Программа предназначена для трансляции учебного конфигурационного языка в формат TOML.
Она принимает входной файл с описанием на учебном языке и формирует выходной файл в формате TOML.
При наличии синтаксических ошибок программа сообщает об этом с указанием строки и столбца.

1)Эта программа переводит текст с учебного конфигурационного языка в формат TOML.
- В папке должны лежать два файла:
- translator.py — сама программа.
- input.conf — входной файл с текстом.
2) Запускать нужно через командную строку. Сначала перейти в папку с файлами
3) Потом выполнить команду:
python translator.py -i input.conf -o output.toml
4) В результате появится файл output.toml с преобразованным текстом.
Если в исходном файле есть ошибки, программа напишет сообщение об ошибке с указанием строки и столбца.
